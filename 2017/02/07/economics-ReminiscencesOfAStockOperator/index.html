Economics-《股票大作手回忆录》

发表于 2017-02-07 | 分类于 Economics | |

《股票大作手回忆录》是一本人物传记，描述了金融投机奇才-杰西·利弗莫尔(Jesse Lauriston Livermore)的传奇人生经历和投资智慧。

股票作手回忆录（全译版）

【美】埃德温·利非弗; 杰西·利弗莫尔


第1章 不要问为什么，原因总比机会迟到很久


2017-02-05 15:33:46

股市上今天发生的事，过去曾经发生过，将来也会再次发生。


2017-02-05 15:37:49

你和报价器的关系，就在当时当下，而不是明天。原因可以以后再说，而现在，你要么立刻行动，要么丢掉机会


第2章 价格是最浅的表象，对涨跌下注是小孩子的游戏


2017-02-05 16:19:39

市场信号出现之前，不要出手。


第3章 所有的错误都要一一犯过，每一个经验都需要真金白银


2017-02-05 18:44:33

人们说，凡事皆有两面性，但股市只有一个面，不是牛面或熊面，而是正确的一面。这是基本原则。


2017-02-05 18:46:45

没人可以给我任何一个或一套建议，依靠我自己的判断才能帮我赚更多的钱。这是个聪明人的游戏


2017-02-05 18:54:00

报价器上的价格和实际价格的巨大差异，让我输得一塌糊涂。


2017-02-05 18:55:11

绝对不能在股价的短期波动上下注。我应该把目光放远，预测市场的整体趋势。


第4章 知道什么不该做，比知道什么该做更加重要


2017-02-05 19:17:30

吃一堑，长一智。当你一无所有，就会恍然大悟，知道什么不该做。当你知道有些事不该做否则会亏，你才能开始知道什么事该做才能赢。


第5章 没人能跟对所有的波动，你笑到最后即可


2017-02-05 19:21:47

波浪线能帮助可以读懂它们的人，或者说，能帮助那些能吸收其信息的人。但一般人读波浪线，会痴迷其顶部、底部、大波段和小波段之类的，他们认为那就是股票投机的全部。如果一个人完全靠这些做操作，信心越爆棚，死得就越快。


2017-02-05 19:22:46

赌涨跌和预测涨跌是截然不同的，这也是赌博和投机最本质的区别。


2017-02-05 19:28:48

赚大钱不能靠细碎的波动，而要靠大走势，也就是说，不能靠解读波浪线，而要抓住基本面。


2017-02-05 19:29:41

读盘能力从没为我赚过大钱。为我赚大钱的策略总是“坚持不动”。


2017-02-05 19:30:56

持对仓位并坚持持仓。


第6章 孤独是投机商的宿命，大势是唯一可靠的盟友


2017-02-05 23:52:46

不要相信投资专家的建议，不要相信所谓的内幕消息，不要相信券商的推荐，不要相信“至少不会损失”的鬼话，你只能相信自己的研究。——彼得·林奇


第7章 作为一个投机商，我从不抄底，也从不逃顶


2017-02-06 00:51:11

应该先研究环境，谨慎地做操作计划，然后再付诸实践。


2017-02-06 00:53:18

在跟着大势做交易的时候，股价永远不会高到你不能买进，也永远不会低到你无法卖空。第一笔交易后，除非有利润，否则不要做第二笔。应该持股观望一阵。这时读盘能力可以帮你判断时机是否真的到来，可以开始操作了。很多事情是否成功，首先要看是否在对的时间行动。


第8章 在错误的时间做正确的事情，是最要命的


2017-02-06 00:56:09

所谓偏见，就是一愿意做空，那么满眼就都是熊市线索；而做多对自己有利时，那么他看到的就都是多头信息。我只有一个偏见，那就是不犯错。


2017-02-06 00:56:37

只有在牛市的整体上扬中才能赚大钱。不管牛市的导火索是什么，它的持续必须依赖基本的经济基础，基金和金融集团的炒作永远无法使其长久。而且，不管是谁从中作梗，都无法阻止大盘按照既定的方向快速持续地走下去，至于能走多远，那就由经济基础的推动力决定了。


2017-02-06 00:59:51

当你读透了大环境，你的交易就不会限制在哪支或哪几支股票上，你可以通盘买进或抛出


2017-02-06 01:14:49

我确定自己错在做得太早了，但我当时实在抑制不住进场的冲动。后来市场又开始下跌，机会来了。我全力做空，可没料到价格再次反弹，而且弹到很高的价位


2017-02-06 01:14:22

我预测对了，却输得精光。


第9章 先做对的事，赚钱只是水到渠成的事情


2017-02-06 01:36:30

我做对了：我不仅看得长远，而且制订并遵循了明确的规划。我明白了要赚大钱，必须怎么干。自此以后，我永远脱离了靠赌个股涨跌过活的日子，而是终于学会了在更高层面上更智慧地做交易。这是我一生中最重要的一天。


第10章 有人性的人做股票，本身就是个错误


2017-02-06 01:40:26

确定两点：首先是多头还是空头，其次是进场的时间。这些在股票和期货市场都完全相同，不管是棉花、小麦、玉米还是燕麦。


2017-02-06 01:43:30

你需要确定现在你手上这支期货的最小阻力方向，也就是静待市场明确地告诉你它已突破最大阻力点，盘整期已过，而且方向明确。这才是你应当进场的唯一指示牌。


2017-02-06 01:47:27

如果价格窄幅波动，波幅小到不值一提，试图预测它的涨跌是毫无意义的。你应该继续等待，观察市场，解读行情以确定波动的上下限，并下定决心，除非价格突破阻力点，否则绝不出手。交易商没必要强求走势按他的判断发展，一切要以赚钱为目的。永远别和行情吵架，也别指望行情给你理由或解释。事后给证券市场解剖验尸的人不会获得任何利润。


2017-02-06 01:49:37

我只是先确定价格的走向，然后不停地施测，看自己是否判断正确，以及确定何时平仓。所谓施测，就是在确立仓位后，试探市场，观察价格的反应。


2017-02-06 01:54:35

投机商最大的敌人往往是他内部的自己。人类有希望与恐惧的本性。


2017-02-06 01:55:36

交易商必须克服这两种根深蒂固的本性，彻底改变冲动的人性，才能在投机市场获得成功。他必须逆转自己人性中的冲动，在普通人希望时恐惧，在普通人恐惧时希望。他必须害怕损失变大，必须希望利润增加。一个正常人进入股市，本身就是个错误。


第11章 视角不同是专业和业余之间唯一的区别


2017-02-06 13:30:29

业余人士认为自己只想兼职赚点钱，所以自以为是，思考不深刻、不透彻；专业选手则只求做正确的事情，这比赚钱更重要，因为他们知道，只要做对了事，利润会是水到渠成的。


第12章 稳如泰山即卓尔不凡，坚持独立思考


2017-02-06 14:11:29

一个人犯蠢是不需要理由的。


2017-02-06 14:12:03

对交易商来说，另一个致命的敌人，就是聪明的朋友的热切规劝和人格魅力。


2017-02-06 14:16:11

赌博心态带来的危险，远非用脑子分析错时可比。


第13章 市场并不奖励忠肝义胆，致命的人情羁绊


2017-02-06 22:35:10

如果一个股票商能学会避免脑子发热，花多少代价都是值得的。很多聪明人犯的严重错误，大抵都可以归于此类。


2017-02-06 22:46:24

人情债有时候是非常昂贵的，而且要用一辈子去还。


2017-02-06 22:47:51

情感再次战胜了理智，我屈服了。我让自己的判断臣服于他的意志。这成了我毁灭的开始。


第14章 股市中没有常胜将军，没有一个股神没做过股渣


2017-02-06 23:10:17

除非赚到足够的本钱，否则判断力再好也没什么用。没有足够的本金，做交易就无法淡定冷静，这种精神状态源自能承受一定损失的能力。


2017-02-06 23:43:49

六周耐心的等待，最终，理智战胜了贪婪和希望。


2017-02-06 23:47:50

市场不会灵光一闪就直冲顶点而去，也不会没有征兆就突然陡转直下跌到舱底。市场可能而且经常在股价开始普遍下跌前很久就已经不再是牛市了。我期待已久的信号来了，我注意到领涨股一个接一个地从最高点回踩了几个点，而且数月以来第一次再也没有涨回去。很明显，它们涨势已尽，所以我必须调整交易策略了。


2017-02-06 23:51:31

绝对不要妄图在最高价抛出。那是蠢人干的活。如果没有上涨空间了，就在回档的第一时间抛出。


第15章 股战不是人与人的争斗，而是眼光与眼光的抗衡


2017-02-07 00:06:50

交易所管理委员会在毫无预警的情况下下令改变交易规则，搞得我深受其害


2017-02-07 00:06:59

在投机操作中，没有什么事是绝对的。


第16章 投资商和投机商不是一个物种，找到最确切的内幕


2017-02-07 00:22:40

投资商和投机商不是一个物种。


第17章 行情第六感：我们知晓的，比我们能说出的多得多


2017-02-07 10:36:02

一个交易商要成功，只能依靠观察、经历、记忆和数学能力。


2017-02-07 10:36:06

他必须在可能性上下注，也就是说，必须预测事情的发展。多年的游戏实践、持续的钻研和不断积累的记忆，让交易商能瞬间做出反应，无论发生的事情在意料之外还是意料之中。


2017-02-07 10:36:21

聪明的商人从来不会停止研究总体环境，以跟上世界的发展，而这些发展不知道会以什么形式在什么时候影响各板块的市场走势。在这行打拼多年后，一个人就会养成耳聪目明的习惯。他的操作几乎都是自动的。这样他就有了专业人士应有的态度，而这件无价之宝能让他时不时地就赢一把！这就是专业和业余（或偶尔）选手之间的不同，这种差别就是天上地下的差别。


2017-02-07 10:43:20

商品市场的动向，是期货商的最佳向导，必须遵从。它之于期货商，就像病人的体温、脉搏、舌苔和眼球的颜色之于医生一样。


2017-02-07 10:45:25

市场吸货这么散，这么慢，而且一卖出就跌这么多，这就等于告诉我说，市场没有购买力。


2017-02-07 10:47:44

好多次，我在确定无疑的牛市买了一支股，但发现同板块中的其他股票并没有表现出涨势，我就会清空。


2017-02-07 10:53:05

内线没有买进。而在牛市不买进自己的股票，他们必然有充分的理由。


2017-02-07 10:54:08

研究板块表现的重要性


第18章 交易商的勇气，就是有信心按照既定策略坚持交易


2017-02-07 11:04:05

投机商的勇气就在于有信心按照自己的决定行动。对我来说，只有在我被证明错了之后才会认为自己错了，所以我不怕犯错。


第19章 华尔街是个造神的地方，一代代股神起起落落


2017-02-07 11:14:06

研究人性总能让人受益匪浅：人类会多么容易轻信自己愿意相信的事情？人类为何总是允许自己（其实是鼓励自己）受贪欲和愚蠢的左右并付出昂贵的代价？


2017-02-07 11:14:11

投机成功的基础原则是：人们将来会和他们过去一样犯相同的错误。”


第20章 操盘手控盘的细节：如何揉捏价格的涨跌造势


2017-02-07 13:41:03

把股价炒热，然后在跌势中散给大家。


2017-02-07 13:48:15

在高价位时，也就是交易员和股民的买进需求旺盛时，我攒了一些空仓，所以现在实际上只是在回补当时的空仓。我会让交易员和股民明白，总有人在股价下跌时买进。如果没有支撑力量股票就会变得越来越软，大家就会纷纷卖出，而我的这种操作既可以阻止交易员们鲁莽放空，也可以阻止恐慌的股民急着卖出。这种回补操作，我称其为“维稳工序”。


第21章 止赢和止损同样重要，成功时多走一步就会变成烈士


2017-02-07 13:59:56

不和报价器叫板，不向烂市场发火。


2017-02-07 16:28:41

普通人开始蜂拥进场，他们只考虑价格而不是价值，只根据情绪行动而不是大环境，采取了最简单的办法，拒绝思考涨势总有尽头。这就是业余人士虽然知道不应在顶点买进，实际上却总买在最高价所以赔得一塌糊涂的原因。于是，民众在繁荣期获得的账面利润，永远停留在了账面上。


第23章 内线绝不会向世界宣布任何事实，操盘手成为猎物


2017-02-07 16:55:04

无知、贪婪、恐惧和希望，所有这些都是人类这种动物的本性，都会让人亏损，而世界上所有的法律和证交所的所有规则都无法消除。


2017-02-07 16:58:38

内线知道公司业务不佳，收入减少，所以在业务好转之前不敢支撑。而一旦业务再次好转，内线又会低调买进。


第24章 内线只会告诉你何时买，但一定不会告诉你何时卖


2017-02-07 17:05:42

在场外买，价格一般比市价低，还可以在自由市场正式卖出，这就足以诱使人们超额买进了


2017-02-07 17:05:29

有一种销售手段，让不肯动脑子的大众亏了几百万，但它完全合法，所以上当了也没法起诉任何人。这种手段就是，由于市场需要所以分拆。


2017-02-07 17:06:38

大家应当时刻谨记股票交易的基本原则。涨了，别问为什么涨，自然是因为持续的买盘。只要股价接着涨（偶尔的小幅回踩属合理现象），继续买进就是稳妥的操作。经过长期的持续上涨后，价格突然回档，逐渐开始下跌，其间偶尔小幅反弹，这时你就明白，方向显然已经从上涨变成下跌了。就是这么简单，


2017-02-07 17:07:42

有人有时能打败某支股票，但没人能永远打败整个市场。无论你经验多么丰富，任何人都可能会输，因为投机不可能百分之百全中
