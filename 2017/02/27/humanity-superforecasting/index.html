Humanity-《超预测》

发表于 2017-02-27 | 分类于 Humanity | |

预测未来的艺术和科学。

乐观的怀疑论者:为什么我们的世界需要预测

可预测性存在难以逾越的局限性。不是每一只巴西的蝴蝶扇扇翅膀，都能引起德克萨斯州的龙卷风。

某事物的可预测性如何，取决于三个因素:我们想要预测的内容，预测的时间远近，以及在什么情况下进行预测。

在预测领域，要抓住机会，只需要做到:一是树立一个清晰的目标，即准确性;二是指定严格的评估方法。

统计算法和机器学习，在预测领域，也许将逐渐取代人类的主观判断，例如医生的诊断。

知识的错觉:糟糕的预测与糟糕的决定

怀疑的精神和科学的严谨。

现代心理学家通常采用双系统模型，该模型将我们的精神世界分为两个系统。第一系统是我们熟悉的思维领域，由我们选择的所有关注点构成。第二系统则不为人熟知，属于无意识感知和认知活动。第一系统往往先启动，得去直觉的答案，然后第二系统才介入，审视答案。

人们天生倾向于抓住第一个貌似合理的解释，然后愉快地搜集支持这个解释的证据，却不会去验证其可靠性。

一种评分系统:超预测是可以衡量的

分析预测的有效成分和无效成分的第一步就是评估预测，预测的含义不能是模糊的，必须是确定的。

布莱尔得分:评估预测的内容和实际情况之间的差距，越低越好，完美得分为0，五五开为0.5，最高为2.0。

集体预测的效果如何，取决于预测的内容是什么，以及参与人是否博闻强记。

超级预测家:准确预测靠的更多是能力而不是运气

均值回归是检验运气对成绩的影响的必要工具，缓慢回归说明主要是能力导致，快速回归则说明是运气主导。

超级预测家整体表现出来的是缓慢回归。

超级聪慧？内部视角、外部视角与其他视角

“外部视角”：宏观，”内部视角”：微观。

要求人们假设他们的初始判断是错误的，然后认真思考错误原因，接着做出新的判断，产生第二次预测，并与第一次相结合，仅仅做到这几点，就能大幅提高准确性。

主动开放思维:

  • 人们应该考虑与自己观念相悖的论据;
  • 关注与你意见不同的人，比关注与你意见相同的人更有用;
  • 改变想法是软弱的变现;
  • 直觉是指定决策最好的指引;
  • 坚定信念，即便是不利证据出现在你的面前，也要始终不渝，这一点很重要。

超级宽客？像华尔街的数字巫师一样

概率具有根深蒂固的反直觉天性，这解释了非常老练的人也经常犯下低级错误的原因。

对不可简化的不确定性的认知是概率思维的核心。

预测者越倾向于命中注定思想，预测的准确率越低，而越倾向于概率思维，准确率越高。

超级新闻迷？如何像蜻蜓的复眼一样敏锐

将解决问题的过程分解为若干部分。尽可能明确区分已知信息和未知信息，认真考虑所有的假设。

借鉴外部视角，把问题置于比较视野下，淡化其特殊性，同时将其作为广泛的同类现象的特例进行处理。然后，通过内部视角分析问题的特殊性。

此外，还要研究自己的观点与其他人的观点有哪些异同，特别要注意预测市场和其他吸取群体智慧的方法。

综合所有视角，形成自己的判断，要像蜻蜓的复眼一样敏锐。

永远的贝塔:超级预测家是如何炼成的

“成长型思维模式”:相信自己的能力主要是努力的结果，自然可以通过“成长”达到愿意努力工作和学习的状态。

总是在颠覆，愿意尝试，向失败学习。分析，跳转和总结。

超级团队:从愚蠢团队到智慧团队

团队的双重特性：团队有可能产生可怕的错误，也可能做出敏锐的判断，并且共同完成仅靠个体无法完成的工作。

团队越多样化，搜集信息的效率越高，团队掌握的信息量是个人无法比拟的，每个成员都会带来与众不同的视角。

培养团队的共享文化:鼓励队友怀着尊重对方的心态相互质疑，勇于承认无知，大胆请求帮助。

领导的两难选择:如何成为超级领导者

领导者必须具有理性的自信心，而且要想被领导者灌输自信。决断力是另一个至关重要的品质，领导者不能无休止地思考。他们必须评估形势，做出决定，付诸行动。第三点，领导者必须会描述蓝图，也就是阐述大家一起努力要实现的目标。

决策由于时间问题，很多时候并非是完美的，但是必须及时，不完美但很及时的决策要好过决策完美但不及时。

一旦决策确定，思维模式要随着转换，忘记不确定性和复杂性，全力投入行动。

在做到高度自信的同时，也要保持明智的谦虚，必须认识到好的预测需要认真思考，而对自身能力的信心又会激励其行事果断。

真的如此出类拔萃?黑天鹅事件也可以预测

“视野迟钝”:人们会在无意识的情况下先入诱惑与转换的陷阱中，回避需要根据时间框架来调整概率的难题，转而处理关于正反两方论据的相对权重的简单问题。

通过巨大努力，至少在某些真正重要的发展趋势上，人们可以得出准确的预测。

接下来做什么?我们可以做得更好

预测的唯一目标应该是准确性，但是在现实中，准确性通常只是诸多目标中的一项，有时甚至无关紧要。当涉及到利益划分时，有时预测还会招惹到非议。在嘈杂的公共领域，争辩各方完全可能也没有兴趣进行对抗性合作。

我们要做的就是认认真真地给预测评分。

给志向远大的超级预测家的“十诫”

  1. 鉴别分类
  2. 将看似棘手的问题分解成容易处理的子问题
  3. 正确平衡内部视角和外部视角
  4. 正确平衡对证据的反应不足和过度反应
  5. 寻找每个问题中发生作用的相互冲突的因果之力
  6. 按照问题自身特性划分尽可能多的不确定性登记，但要适可而止
  7. 正确平衡信心不足与过分自信、慎重与果断
  8. 探寻错误背后的问题，但要谨防后视偏差
  9. 激发他人的最大潜能，同时也让他人激发你的最大潜能
  10. 掌握骑自行车的技术
  11. 活学活用
