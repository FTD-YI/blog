Python 基础知识。

定义

Python 是一门解释型的高级语言(在执行时一行行翻译为机器码)，其语法简单优雅，但是在速度上不及编译型语言(例如 C)，另外，无法加密。

Python 内置了完整的基础库，覆盖网络、文件、GUI、数据库、文本等，另外还有大量的第三方库，因此开发速度会非常快。

解释器

CPython

官方的解释器，基于 C 语言，会将代码转换成字节码。

PyPy

PyPy 是一个 JIT 解释器，其会直接编译成机器码，而非字节码。
